# Forum
## Simple discussion forum<br/>
https://ntp-forum.herokuapp.com/

## Technologies:
### FrontEnd:
1. React Js
2. Draft js (rich editor)
3. Redux
4. Sass modules
5. Formik
5. Axios
 ### BackEnd:
1. Express Js
2. MongoDB cloud + Mongoose
3. JsonWebToken
4. helmet + bodyParser
